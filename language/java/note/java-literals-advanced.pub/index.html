    Java мӢ¬нҷ”: лӢӨм–‘н•ң лҰ¬н„°лҹҙ н‘ңнҳ„мӢқ¶

    ========

    л¬ёмһҗ лҰ¬н„°лҹҙ

    л¬ёмһҗм—җ лҢҖн•ҙ мҪ”л“ңк°’мқ„ м •мқҳн•ң н‘ң = Character Set

    ASCII(7bit)¶

     A -> 100 0001
 B -> 100 0010
 C -> 100 0011
 ...
 мҳҒм–ҙ лҢҖмҶҢл¬ёмһҗ + нҠ№мҲҳл¬ёмһҗ м•Ҫ 90м—¬к°ң

    ISO-8859-1(8bit)¶

     ASCII + м„ңмң лҹҪл¬ёмһҗ = 200м—¬к°ң
 A -> 0100 0001
 B -> 0100 0010
 C -> 0100 0011
 ...

    EUC-KR(16bit)¶

     н•ңкёҖ 2350мһҗ
 мҳҒм–ҙ : ISO-8859-1
 н•ңкёҖ : 2byte мҪ”л“ңк°’
 к°Җ -> 0xB0A1
 к°Ғ -> 0xB0A2
 лҳҳ -> 0xB6CA
 лҳ  -> м •мқҳлҗҳм–ҙ мһҲм§Җ м•ҠмқҢ
 лҳЎ -> м •мқҳлҗҳм–ҙ мһҲм§Җ м•ҠмқҢ
 лҳЈ -> м •мқҳлҗҳм–ҙ мһҲм§Җ м•ҠмқҢ
 лҳҘ -> 0xB6CB

    мЎ°н•©нҳ•(16bit)¶

     л§ЁмҷјмӘҪ1л№„нҠё + мҙҲм„ұ 5л№„нҠё + мӨ‘м„ұ 5л№„нҠё + мў…м„ұ 5л№„нҠё = 16л№„нҠё
 м•Ҫ 3л§Ңмһҗмқҳ н•ңкёҖмқ„ н‘ңнҳ„н•  мҲҳ мһҲмңјлӮҳ көӯм ңн‘ңмӨҖмқҖ м•„лӢҳ

    MS949(11172мһҗ)¶

     EUC-KR + м•ҢнҢҢ
 windows95 ~ м§Җмӣҗ
 лҳҳ -> 0xB6CA
 лҳ  -> 0x8C63
 лҳЎ -> 0x8C64
 лҳҘ -> 0xB6CB
 көӯм ңн‘ңмӨҖмқҙ м•„лӢҳ
 нҳ„мһ¬ мҡ°лҰ¬к°Җ мӮ¬мҡ©н•ҳлҠ” мңҲлҸ„мҡ° м»ҙн“Ён„° лӘ…л №м–ҙм°Ҫм—җм„ңлҠ” MS949мӮ¬мҡ©

    мң лӢҲмҪ”л“ң¶

    • көӯм ң н‘ңмӨҖ мқёмҪ”л”© к·ңм№ҷмқҙлӢӨ.
    • UCS-2мқҳ кІҪмҡ° мҳҒм–ҙ, н•ңкёҖ лӘЁл‘җ 2л°”мқҙнҠёлЎң мқёмҪ”л”©н•ңлӢӨ.
    • MS949мқҳ лӘЁл“  н•ңкёҖ л¬ёмһҗк°Җ нҸ¬н•Ёлҗҳм–ҙ мһҲлӢӨ.
    • A(0x0041), B(0x0042), +(0x002b), к°Җ(0xac00), к°Ғ(0xac01), ...
    • кё°мЎҙмқҳ EUC-KRкіј MS949 мҪ”л“ңмҷҖ нҳёнҷҳлҗҳм§Җ м•ҠлҠ”лӢӨ.(н•ңкёҖмҪ”л“ң мғҲлЎң м •мқҳ)
    • JVMмқҖ лӮҙл¶Җм—җм„ң л¬ёмһҗм—ҙмқ„ лӢӨлЈ° л•Ң UCS-2лҘј мӮ¬мҡ©н•ңлӢӨ. мҰү мҳҒм–ҙ, н•ңкёҖ лӘЁл‘җ 2л°”мқҙнҠё мң лӢҲмҪ”л“ңлЎң лӢӨлЈ¬лӢӨ.

    UCS(Universal Coded Character Set; көӯм ң л¬ёмһҗ м§‘н•©)¶

    • 'ISO 10646' н‘ңмӨҖмқ„ к°ҖлҰ¬нӮӨлҠ” мқҙлҰ„мқҙлӢӨ.
    • UCSлҠ” 110л§Ңк°ң мқҙмғҒмқҳ л¬ёмһҗм—җ лҢҖн•ҙ мҪ”л“ң к°’мқ„ м •мқҳн•ҳкі  мһҲлӢӨ.
    • к·ё мӨ‘м—җм„ң мІ« 65536к°ң(0 ~ 65535 к№Ңм§Җмқҳ мҪ”л“ң)мқҳ л¬ёмһҗ мҪ”л“ңк°Җ мЈјлЎң лӢӨ.
    • мқҙ лІ”мң„мқҳ мң лӢҲмҪ”л“ңлҘј 'BMP(Basic Multilingual Plane; кё°ліё нҸүл©ҙ)'лқј н•ҳл©° ліҙнҶө 'UCS-2'лқј л¶ҖлҘёлӢӨ. мҰү 2л°”мқҙнҠё лІ”мң„м—җм„ң м •мқҳн•ң мң лӢҲмҪ”л“ңлқјлҠ” мқҳлҜёлӢӨ.
    • мһҗл°”к°Җ мӮ¬мҡ©н•ҳлҠ” мң лӢҲмҪ”л“ңлҠ” л°”лЎң мқҙ 'UCS-2'лқј л¶ҲлҰ¬лҠ” лІ”мң„мқҳ л“ңмқҙлӢӨ. кё°мЎҙ EUC-KR, ISO-8859-1мҷҖ нҳёнҷҳX

    UTF-8¶

    • UCSмқҳ мҪ”л“ң к°’ мӨ‘м—җм„ң 00 ~ 7F(127к°ң)к№Ңм§Җ ASCIIм—җ н•ҙлӢ№н•ҳлҠ” UCS к·ёлҢҖлЎң 1л°”мқҙнҠёлЎң н‘ңнҳ„н•ңлӢӨ.(мҳҒм–ҙлҘј мҳҲм „мІҳлҹј 1byteлЎң) л”°лқјм„ң ASCIIлҘј кё°ліёмңјлЎң мӮ¬мҡ©н•ҳлҠ” мӢңмҠӨн…ңмқҳ кІҪмҡ° UTF-8лЎң мқёмҪ”л”© мқҙн„°лҘј нҠ№лі„н•ң мІҳлҰ¬м—Ҷмқҙ к·ёлҢҖлЎң мқҪкі  м“ё мҲҳ мһҲлӢӨ.
    • к·ё мҷёмқҳ л¬ёмһҗ мҪ”л“ңлҠ” к·ңм№ҷм—җ л”°лқј 2л°”мқҙнҠё ~ 4л°”мқҙнҠёк№Ңм§Җ ліҖнҷҳн•ҳм—¬ лӢӨ.
    • UCSмқҳ UTF-8 ліҖнҷҳ к·ңм№ҷ 000000 ~ 00007F: 0xxxxxxx
      000080 ~ 0007FF: 110yyyxx 10xxxxxx
      000800 ~ 00FFFF: 1110yyyy 10yyyyxx 10xxxxxx
      100000 ~ 10FFFF: 11110zzz 10zzyyyy 10yyyyxx 10xxxxxx
    • лҢҖл¶Җл¶„мқҳ мӢңмҠӨн…ңм—җм„ң лҚ°мқҙн„° мһ…м¶ңл Ҙ н•  л•Ң UTF-8мқ„ мӮ¬мҡ©н•ңлӢӨ.
    • мһҗл°”м—җм„ңлҸ„ лҚ°мқҙн„° мһ…м¶ңл Ҙн•  л•Ң мЈјлЎң UTF-8мқ„ мӮ¬мҡ©н•ҳкі  мһҲлӢӨ. н•ңкёҖмқҖ 3byte (мң лӢҲмҪ”л“ңм—җм„ңлҠ” 2byte)

 лӘЁл“  л№„нҠёлҘј л°ҳм „. 0мқ„ 1лЎң 1мқ„ 0мңјлЎң

 2's complement(2мқҳ ліҙмҲҳ)
 1мқҳ ліҙмҲҳлЎң м ҖмһҘлҗң мқҢмҲҳ к°’мқ„ лҚ”н•  л•Ңл§ҲлӢӨ кі„мӮ° кІ°кіјм—җ 1мқ„ м¶”к°Җн•ҳлҠ” лІҲкұ°лЎңмӣҖмқ„ м—Ҷм• кё° мң„н•ҙ
 лҜёлҰ¬ мқҢмҲҳлҘј м ҖмһҘн•  л•Ң лҜёлҰ¬ 1мқ„ м¶”к°Җн•ҙл‘җлҠ” л°©лІ•
 => 1мқҳ ліҙмҲҳ +1 => 2мқҳ ліҙмҲҳ
 1111 1111 1111 1111 1111 1111 1110 1001 (-23)
 0000 0000 0000 0000 0000 0000 0001 0111  (23)
 ---------------------------------------
10000 0000 0000 0000 0000 0000 0000 0000 = 0
 к·ёлһҳм„ң м»ҙн“Ён„°м—җм„ң мқҢмҲҳлҘј л©”лӘЁлҰ¬м—җ м ҖмһҘн•  л•ҢлҠ”
 м–‘мҲҳмҷҖ мқҢмҲҳлҘј лҚ”н•  л•Ң м •мғҒм Ғмқё к°’мқҙ лӮҳмҳӨлҸ„лЎқ

4л°”мқҙнҠё л©”лӘЁлҰ¬м—җ м ҖмһҘн•  мҲҳ мһҲлҠ” л¶ҖлҸҷмҶҢмҲҳм җмқҖ мҶҢмҲҳм җмқҳ мң„м№ҳм—җ мғҒкҙҖм—Ҷмқҙ 7мһҗлҰ¬(мң нҡЁмһҗлҰҝмҲҳ)

к·ёлһҳм„ң л¶ҖлҸҷмҶҢмҲҳм җмқҖ л©”лӘЁлҰ¬м—җм„ң кәјлӮё к°’мқ„ лӮҙл¶Җмқҳ к·ңм№ҷм—җ л”°лқј JVMмқҙ ліҙм •н•ҙмӨҖлӢӨ.
