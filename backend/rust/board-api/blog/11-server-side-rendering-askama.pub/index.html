    лёҢлқјмҡ°м Җмҡ© кІҢмӢңнҢҗмқҖ JSON лҢҖмӢ  HTMLмқ„ л°ҳнҷҳн•©лӢҲлӢӨ. board-apiлҠ” AskamaлЎң Rust structмҷҖ .html н…ңн”ҢлҰҝмқ„ м»ҙнҢҢмқј нғҖмһ„м—җ л¬¶мҠөлӢҲлӢӨ (Jinja2/Thymeleaf, Django templateкіј мң мӮ¬).

    RustлҘј мІҳмқҢ м ‘н•ңлӢӨл©ҙ вҖ” SSRмқҙлһҖ¶

    SSR(Server-Side Rendering) вҖ” м„ңлІ„к°Җ HTML л¬ёмһҗм—ҙмқ„ мҷ„м„ұн•ҙм„ң лёҢлқјмҡ°м Җм—җ ліҙлғ…лӢҲлӢӨ. React SPAмІҳлҹј лёҢлқјмҡ°м Җк°Җ JSONмқ„ л°ӣм•„ нҷ”л©ҙмқ„ к·ёлҰ¬лҠ” л°©мӢқмқҙ м•„лӢҷлӢҲлӢӨ.

    • мһҘм җ вҖ” JavaScript м—ҶмқҙлҸ„ лҸҷмһ‘, SEOВ·мІ« нҷ”л©ҙмқҙ лӢЁмҲң.
    • мқҙ н”„лЎңм қнҠё вҖ” нҸј м ңм¶ң вҶ’ м„ңлІ„ кІҖмҰқ вҶ’ лҰ¬лӢӨмқҙл үнҠё лҳҗлҠ” м—җлҹ¬ HTML. м „нҳ•м Ғмқё вҖңнҒҙлһҳмӢқвҖқ мӣ№ м•ұ нқҗлҰ„мһ…лӢҲлӢӨ.

    fn render_page(title: &str, body: impl Template) -> Result<Html<String>, AppError> {
    let body_html = body
        .render()
        .map_err(|err| AppError::Validation(format!("template error: {err}")))?;
    let page = LayoutTemplate {
        title,
        body: &body_html,
    };
    let html = page
        .render()
        .map_err(|err| AppError::Validation(format!("template error: {err}")))?;
    Ok(Html(html))
}
